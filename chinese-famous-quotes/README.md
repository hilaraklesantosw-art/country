# Chinese Famous Quotes

一个可直接接入产品的中文名人名言小型开源库。

## 目标

这个库用于支撑以下场景：

- 每日金句
- 开屏一句
- 励志卡片
- 任务提醒文案
- AI 助手的陪伴式表达

## 设计原则

- 优先使用中文世界高识别度作者
- 优先收录短句、强语义、可独立使用的表达
- 尽量避免过长段落，方便在卡片、推送、组件中复用
- 字段结构简单，便于前后端或推荐系统直接接入

## 数据文件

- `data/quotes.zh-CN.json`

## 字段说明

- `id`: 唯一 ID
- `quote`: 金句内容
- `author`: 作者
- `source`: 典籍或作品来源
- `era`: 时代
- `themes`: 主题标签
- `tone`: 情绪色彩
- `usage`: 推荐使用场景

## 使用建议

如果你要做“个性化每日金句”，推荐把这份库作为中文候选池，再结合：

- 用户基础信息
- 今日运势标签
- 近七天对话记忆关键词
- 当日日程标签

做召回和排序。

## JSON 示例

```json
{
  "id": "cq_001",
  "quote": "学而不思则罔，思而不学则殆。",
  "author": "孔子",
  "source": "《论语·为政》",
  "era": "春秋",
  "themes": ["学习", "思考", "成长"],
  "tone": "理性",
  "usage": ["学习激励", "自我提升", "晨间提醒"]
}
```

## 许可说明

本目录中的结构化整理、标签设计与示例数据编排按仓库整体使用方式开放。所收录内容优先选择古代典籍中的常见短句，适合做公开语料底稿与产品原型验证。
