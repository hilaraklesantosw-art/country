# 写作风格指南

## 通用原则

### 清晰简洁
- 避免冗长的句子
- 使用主动语态
- 删除不必要的修饰词

### 专业得体
- 术语使用一致
- 避免俚语（在学术/技术写作中）
- 保持客观中立

### 便于阅读
- 适当使用列表和表格
- 重要内容加粗或斜体
- 保持段落长度适中（3-5句）

## 各类型书籍风格

### 学术/技术类
- 严谨、准确
- 大量使用专业术语
- 注重数据和证据
- 引用可靠来源

### 通俗/入门类
- 通俗易懂
- 少用专业术语，必要时解释
- 多用类比和例子
- 保持轻松友好的语气

### 商业/管理类
- 实用导向
- 强调可操作性
- 案例丰富
- 结论先行

### 小说/文学类
- 注重叙事技巧
- 人物塑造立体
- 情节吸引人
- 语言富有感染力

## 常见错误及避免

| 错误类型 | 问题 | 改进方法 |
|---------|------|---------|
| 句子过长 | 理解困难 | 拆分为短句 |
| 术语不一致 | 混淆读者 | 建立术语表 |
| 逻辑跳跃 | 难以理解 | 增加过渡句 |
| 例子不足 | 抽象难懂 | 多举实例 |
| 口语化 | 不够专业 | 改为书面语 |

## 检查清单

初稿完成后检查：

- [ ] 语法正确
- [ ] 拼写无误
- [ ] 标点规范
- [ ] 术语统一
- [ ] 逻辑连贯
- [ ] 格式一致
- [ ] 引用准确
