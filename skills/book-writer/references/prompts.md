# 写作阶段Prompt模板

## 阶段1：确定主题

```
我想写一本关于[主题]的书。
- 目标读者是：[描述读者群体]
- 预计规模：[字数范围]
- 风格：[学术/通俗/技术/小说等]

请帮我明确写作方向和重点。
```

## 阶段2：编写大纲

```
书名：《[书名]》
主题：[简述书籍核心内容]

请帮我编写一份完整的书籍大纲，包含：
- 书籍整体结构
- 章节安排（建议[数字]章）
- 每章的核心主题

格式要求使用Markdown标题层级。
```

## 阶段3：编写详细提纲

```
基于以下大纲：
[粘贴大纲]

请为每一章编写详细的提纲，包含：
- 每章的3-5个小节
- 每个小节的3-5个核心要点
- 各章节之间的逻辑关系说明

要求内容充实，为撰写初稿做好准备。
```

## 阶段4：撰写初稿

```
请根据以下提纲撰写第[数字]章的初稿：
[粘贴该章提纲]

要求：
- 字数约[数字]字
- 风格：[确定风格]
- 格式：Markdown
- 要点：完整、详细、逻辑清晰
```

## 阶段5：润色初稿

```
请对以下章节进行[第一轮/第二轮/第三轮]润色：

[粘贴章节内容]

本轮润色重点：
- [第一轮]结构完整性、内容连贯性
- [第二轮]语言表达、术语统一、说服力
- [第三轮]语法检查、格式统一、细节打磨
```

## 阶段6：整合成书

```
我有以下章节的最终版本：
[章节列表或文件]

请帮我：
1. 检查整体结构和逻辑
2. 添加目录
3. 编写书籍简介/前言
4. 检查格式一致性
5. 提出最终修改建议

最终目标：形成一本完整、专业的书籍。
```
