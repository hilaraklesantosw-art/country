# 《Codex从入门到实战》

副标题：从第一次提问到真实项目交付

这是一本面向中文开发者的 Codex 实战书稿。它不是产品说明书，也不是概念科普集，而是一本围绕“如何把 Codex 真正用进开发工作流”的操作型书籍。

本书默认读者具备最基础的计算机使用能力，但不要求你已经是资深工程师。只要你愿意把 Codex 当成一个需要被正确协作、正确约束、正确验证的开发助手，本书就能为你提供一条完整路径。

## 本书适合谁

- 第一次认真接触 Codex，希望从零建立使用框架的人
- 已经会用聊天式 AI，但还不会把它变成交付工具的人
- 想把 AI 真正纳入前端、后端、调试、测试和项目协作的人
- 想通过高频使用 Codex 提升工作效率的个人开发者、自由职业者和小团队成员

## 本书不解决什么

- 不承诺把读者训练成“零思考全自动开发者”
- 不鼓励把判断责任直接外包给 AI
- 不试图把 Codex 描绘成万能工具
- 不回避会员、效率、商业支持和长期使用成本这些现实问题

## 阅读顺序

如果你是第一次接触 Codex，建议按顺序阅读。

如果你已经有基础，推荐使用下面的跳读路径：

- 想快速开始：第 1-4 章
- 想建立工作流：第 5-6 章
- 想直接上项目：第 7-10 章
- 想规避风险并长期提效：第 11-12 章

## 本书核心承诺

- 帮你完成从“知道 Codex”到“把 Codex 放进真实项目”的跨越
- 帮你形成一套能长期复用的任务模板和验证习惯
- 帮你把 AI 使用从零散提问升级为稳定工作流
- 帮你在保持判断力的前提下，真正把效率提上来
