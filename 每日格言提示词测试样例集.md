# 每日格言提示词测试样例集

更新时间：2026-03-21

本文整理 10 个不同职业、不同情绪压力、不同生活阶段的测试案例，用于验证“每日格言”提示词在以下方面的稳定性：

- 是否只挑选真实存在且有明确来源的句子
- 是否能根据状态自然选择中国或外国经典
- 是否能稳定输出符合约束的 JSON

使用方式：

1. 将你的“每日格言生成提示词”作为固定系统提示词。
2. 将下方任一案例的 `input` 作为用户输入。
3. 检查模型输出是否满足格式、真实性、长度、语气与贴合度要求。
4. 可将 `reference_output` 作为人工评估参考，不要求模型逐字一致，但要求同等质量。

## 建议检查项

- `daily_quote` 是否为真实存在的原句，而非改写或拼接
- `author` 与 `source` 是否明确且不空泛
- `origin` 是否只输出 `"中国"` 或 `"外国"`
- `daily_quote` 是否控制在 8 到 35 个汉字之间
- `reason` 是否简洁，并且真正对应当天状态
